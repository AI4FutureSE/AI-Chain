Gartner forecasts worldwide low-Code development technologies market to grow 20% in 2023. No code AI tools allow non-technical users to create, deploy and manage AI models and applications without the need for coding or programming skills. These tools provide visual interfaces, drag-and-drop components and pre-built templates for creating AI models and workflows, making it easier for users to build and train machine learning models, automate processes, and integrate AI into their existing systems.

No Code AI Landscape 2022 (https://levity.ai/blog/no-code-ai-map)

The above figure, provided by levity.ai (a no code AI tool for robotic process automation), summarizes the 2022 no code AI technology landscape. As we can see, no code AI tools typically provide pre-built models for common AI tasks such as natural language processing, computer vision, and predictive analytics, which can be customized and configured to meet specific needs. The powerful capabilities of large language models have also been integrated into various application scenarios, including education, finance, marketing, gaming, and accessibility. There are also tools such as Microsoft AI Builder and Zaiper, which provide a no code way to embed AI capabilities into workflows. All these no-code AI tools mainly simplify the implementation of AI functionalties, and are typically limited to specific business or robotic process automation scenarios.

To develop AI services for open scenarios, people can generate code through dialogue with ChatGPT, but the dialogue needs to be driven by someone with domain expertise, and the resulting code still needs to be assembled and debugged into actual software applications. Autonomous agent frameworks can autonomously generate software applications or user interface design from natural language, but these frameworks are limited by the LLM's ability to decompose tasks and map sub-tasks to models or other APIs. Therefore, neither ChatGPT nor AutoGPT is a realistic no code tool to handle real-world software production tasks.

Prompt Sapper provides the whole-process support for developing AI services based on foundation models for open scenarios, including but not limited to implementing AI chains through a no-code interface. It stands on the shoulder of LLMs and supports various co-pilots in analyzing, designing, and even preliminarily implementing AI chains in the Exploration view and Design view.
